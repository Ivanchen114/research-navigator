## ✅ 已完成決策

### 1. READ / DO / REF 三層視覺系統
**決策**：全站內容依互動性質分三層，視覺語言必須讓學生不讀文字就知道「我要做什麼動作」。

| 層 | 意義 | 視覺信號 |
|----|------|---------|
| READ | 吸收知識 | 白底/米白，無輸入元素 |
| DO | 主動行動 | 有輸入區/勾選區，底色可辨別 |
| REF | 按需查閱 | DepthBlock 預設收合，灰底等寬字體 |

**依據**：PDF 視覺審查發現學生分不出「這裡要讀」vs「這裡要做」。

---

### 2. ThinkRecord / AIREDNarrative 不得放入 DepthBlock
**決策**：DO 元件（ThinkRecord、AIREDNarrative、Checklist）一律暴露在主流程。DepthBlock 只放 READ/REF 內容（詞彙說明、範例、原則）。

**已修正**：
- W15：自查紀錄、三欄結論鷹架、AI 建議紀錄、判斷取捨紀錄、AI-RED 敘事紀錄
- W14：反思紀錄、盲點紀錄、壓力測試紀錄、AI 協作反思紀錄、AI-RED 敘事紀錄
- W13：原本就無違規

**標籤與顏色**：

### 5. 流程型內容 — Timeline sub-steps 垂直連線格式
**決策**：有 3+ 個依序子步驟的 READ/DO 流程，一律使用 Timeline 格式（左側垂直灰線 + 彩色圓形 badge）。

**視覺規格**：
```jsx
<div className="relative pl-8">
  {/* 垂直連線 */}
  <div className="absolute left-[11px] top-3 bottom-3 w-[2px] bg-[var(--border)]" />
  <div className="space-y-4">
    {steps.map(({ label, desc, accent }, i) => (
      <div key={i} className="relative flex gap-3">
        {/* 圓形 badge */}
        <div
          className="absolute -left-8 w-6 h-6 rounded-full text-white text-[11px] font-bold font-mono flex items-center justify-center z-10 shrink-0"
          style={{ backgroundColor: accent }}
        >
          {i + 1}
        </div>
        <div className="flex-1 bg-white border border-[var(--border)] rounded-[var(--radius-unified)] px-4 py-3">
          <p className="font-bold text-[13px] mb-0.5" style={{ color: accent }}>{label}</p>
          <p className="text-[11.5px] text-[var(--ink-mid)] leading-relaxed">{desc}</p>
        </div>
      </div>
    ))}
  </div>
</div>
```

**accent 色對應慣例（流程語意）**：
| 步驟性質 | 顏色 |
|---------|------|
| 危險/前置（規則訂定） | `#DC2626` 紅 |
| 警戒/處理 | `#D97706` 橘 |
| 定義/設計 | `#2563EB` 藍 |
| 執行/填入 | `#059669` 綠 |
| 驗收/確認 | `#7C3AED` 紫 |

**首次使用**：W13 Step 1「資料整理通用流程（5步驟）」（2026-05-18）

## 🧠 設計原則備忘

- **StepBriefing 是預覽層**：告訴學生這個 Step 有哪幾種類型。ContentTypeChip 是執行層：貼在每個具體 section 上。兩者功能不同，不重複。
- **DepthBlock 的黃金規則**：只放「可跳過也不影響主線任務」的內容。必看概念、ThinkRecord、Checklist 一律不放。
- **「情境概念」vs「純理論」**：情境概念（讀完馬上要用）可以在主流程，但也要貼「學」chip。純理論（背景知識）適合放 DepthBlock 或自學頁。
