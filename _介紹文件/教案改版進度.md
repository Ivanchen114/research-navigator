## 命名規範（兩本「書」不要搞混）

| 名稱 | 性質 | 誰寫 | 哪裡 |
|---|---|---|---|
| **方法工具書** | 跨週速查手冊（reference book） | 老師寫，全班共用 | `/tools/methods`（MethodToolbookPage） |
| **本組工具設計書** | 各組期中產出（design document） | 學生寫，本組獨有 | W11 小組作業評分重點 |

**底層教訓**：兩個都叫「工具書」會口頭混淆（前綴變裝飾）。修法是換掉重複的核心詞，不是加更明顯的前綴。

## 跨週連動鏈（簡圖）

```
W1-W4 探索/定題 → W5-W6 定義/海報 → W7-W8 文獻 → W9-W10 計畫書地基/定稿
   ↓
W11 工具設計書 + Pilot + 倫理（期中報告繳交點）
   ↓
W12 期中短報（同儕把關，正式施測前最後擋板）
   ↓
W13 中期盤點 + 資料整理 → W14-W15 圖表/推論 → W16-W17 海報/發表
```

**關鍵接點**：
- W9 → W10：計畫書 1-5 章 → 整本定稿

- W11 → W12：Pilot 共識合議 → 期中短報
- W12 → W13：同儕回饋 + 老師評分 → 施測收尾 + 原始資料整理（W13 資料整理週）
